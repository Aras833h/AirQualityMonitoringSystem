# Air Quality Monitoring System

## Overview

This project was developed as the final project for the Software Engineering course.

The system monitors air pollution data, calculates the Air Quality Index (AQI), generates reports, and provides alerts when pollution levels become unhealthy.

## Technologies

- Python
- Git & GitHub
- PyTest
- Draw.io
- Visual Studio Code

## Project Structure

```
docs/
src/
tests/
```

## Running the Project

```bash
python src/main.py
```

## Running Tests

```bash
py -m pytest
```

## Test Coverage

```bash
py -m pytest --cov=src --cov-report=html
```
